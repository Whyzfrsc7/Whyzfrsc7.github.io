# Ryouta's Personal Space

Personal space bio website built with Vite, Tailwind CSS, Lenis Smooth Scroll, and integrated with Discord live presence via Lanyard API.

## Features

- **Dynamic Discord Status**: Live avatar, online status indicator, and dynamic custom status speech bubble.
- **Spotify Integration**: Displays active Spotify playback with tracks and artists.
- **Brand Social Grid**: Clean card layout linked to Instagram, Discord, GitHub, and Steam with official brand icons.
- **Smooth Physics Scroll**: Implemented with Lenis for an elegant scrolling experience.
- **Glowing Crosshair Cursor**: Dynamic responsive crosshair custom mouse pointer.

## Technologies Used

- Vite
- Tailwind CSS
- FontAwesome 6 (CDN)
- Lenis Smooth Scroll (CDN)
- Lanyard API
